\subsection{Detailed Description}
A Lora\+Transceiver manages the hardware used for radio transmission with Lo\+Ra protocol. 

\begin{DoxyAuthor}{Author}
F.\+Fargon
\end{DoxyAuthor}
\begin{DoxyVersion}{Version}
V1.\+0
\end{DoxyVersion}
\begin{DoxyDate}{Date}
16/02/2018
\end{DoxyDate}
This class is the interface exposed by objects implementing the low level layer of Lo\+Ra packet transmission (i.\+e. the drivers of Semtech S\+X12xx chips).~\newline
 In current implementation, this interface is provided by the C\+S\+X1276 object.
