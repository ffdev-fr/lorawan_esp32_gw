The ACC client starts a new 'AccConnection' by opening a socket on the 'Command' port and another socket on the 'Event' port of the ACCServer:
  • The sequence must be respected (i.e. 'Command' port first, then 'Event' port).
  • The connection is ready only when both sockets have been opened.
